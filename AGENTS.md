# Aivio Business OS — Codex Instructions

Aivio is NOT a CRM.
Aivio is an AI Business Operating System for SMB.

Core flow:
Message → AI analysis → Contact → Lead/Deal → Task → Finance insight.

All UI text must be in Russian.

Use product sections:
- Центр управления
- Клиенты
- Inbox
- AI-сотрудники
- Автоматизации
- Финансовый радар
- Интеграции
- Шаблоны
- Настройки

Design:
Premium AI minimalism.
No cyberpunk.
No neon.
Inspired by Linear, Stripe, Notion, Apple.

Main integrations:
Telegram, WhatsApp, Instagram, VK, Google Calendar, Google Sheets, Yandex, 1C, МойСклад, Bitrix24, OpenRouter, n8n, Make.

AI employees:
- AI-квалификатор
- AI-саппорт
- AI-аналитик
- AI-финансист
- AI-ассистент руководителя

Industry templates:
- Услуги
- Консалтинг
- Бьюти
- Ритейл
- Онлайн-образование
- Цветочный бизнес

Before finishing:
- run build
- fix TypeScript errors
- do not rewrite the whole app
- make targeted changes
